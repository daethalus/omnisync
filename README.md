# omnisync
How to use:

edit the file config.properties, 

server.url=http://localhost:8500 - the remote url server

http.user=omnisync - http user

http.pass=omnisync - http pass.


in the server side add this mod in the mod folder, the server will open a http server.
in the client use, use the omnisync-install https://github.com/daethalus/omnisync-install/ to pack the .jar and create a installer to the users.
